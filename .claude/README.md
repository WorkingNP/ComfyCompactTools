# .claude ディレクトリ構造

このディレクトリには、Claude Code の設定とプロジェクト固有のカスタマイズが含まれています。

## ディレクトリ構成

### `rules/`
プロジェクト固有のルールとガイドラインを格納します。
- コーディング規約
- テストガイドライン
- アーキテクチャルール
- ワークフロー固有のルール

### `skills/`
カスタムスキル（コマンド）を格納します。
- プロジェクト固有のタスク自動化
- 再利用可能なワークフロー

### `agents/`
カスタムエージェント定義を格納します。
- 専門的なタスクに特化したエージェント設定

### `prompts/`
再利用可能なプロンプトテンプレートを格納します。
- レビュープロンプト
- 生成プロンプト
- 分析プロンプト

## ファイル

- `settings.local.json` - ローカル設定（gitignore推奨）
- `settings.json` - プロジェクト共有設定

## 使い方

各ディレクトリにMarkdownファイルを配置することで、Claude Codeがプロジェクト固有のコンテキストを理解します。
