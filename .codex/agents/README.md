# Agents ディレクトリ

GPT Codex用のカスタムエージェント定義を配置します。

## エージェントとは

特定のタスクに特化した、事前定義された役割・知識・ツールセットを持つCodexの実行モードです。

## エージェント定義の構造

```markdown
# {Agent Name}

## 役割
このエージェントの責務

## 専門知識
- 知識領域1
- 知識領域2

## 使用ツール
- ツール1
- ツール2

## 制約事項
- 制約1
- 制約2

## 典型的なタスク
1. タスク例1
2. タスク例2
```

```

## エージェント例

- `comfy-workflow-builder.md` - ComfyUIワークフロー構築専門
- `test-engineer.md` - テスト作成・実行専門
- `code-reviewer.md` - コードレビュー専門
