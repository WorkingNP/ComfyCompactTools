# VRTask Prompt Set (Flux2)

## Base 1: 小さなVRコワーキング空間の静けさ
_VRChatのような小さな個室コワーキング空間。木製の小デスクとミニマルな椅子、ノートPCに向かうアニメキャラの後ろ姿。時間帯が特定できない穏やかな光、仕事中だが気張りすぎない雰囲気。主役は空気感。_
1. 小さなVRコワーキングの個室、木製の小デスクにノートPC、ミニマルな椅子に座るアニメキャラの後ろ姿。時間のわからない柔らかな室内光で、淡いVRChat空間の空気感が主役。
2. VRChatの小部屋にある木製デスクと簡素な椅子、背中を向けたアニメキャラが軽く作業している。外の時間が読み取れない落ち着いた光、日常的な仕事風景。
3. こぢんまりしたVR空間のコワーキング部屋、ノートPCに向かうアニメキャラの背中。木製デスクとミニマルチェア、ストーリー性は抑えた穏やかな空気感。
4. VRChatっぽい小さな作業部屋、背中を見せるアニメキャラがゆるく仕事中。木の小デスクと簡素な椅子、時間感のないやわらかな照明。
5. バーチャル空間の静かなコワーキング、木製の小さな机とミニマル椅子、ノートPCに向かうアニメキャラの後ろ姿。控えめで日常的な空気。
6. VR空間の小部屋で仕事をしているアニメキャラの背中。木製デスクと簡素な椅子、外の時間がわからない穏やかな光、VRChat的な生活感。

## Base 2: ほんのりネオンとUIの存在感
_小さなVRルームに薄いネオンや半透明UIが漂う。木製の机とミニマル椅子、アニメキャラの背中。集中しすぎず自然体の仕事。_
1. うっすらネオンのあるVRコワーキングルームで、木製の小デスクにノートPC、背中を向けるアニメキャラ。半透明のUIが浮かび、時間帯は曖昧。
2. VRChat的な小部屋、淡いネオンと浮遊UI。木の小さな机とミニマル椅子、アニメキャラの後ろ姿がゆったり作業している。
3. 薄いネオンの反射があるVR空間、ノートPCに向かうアニメキャラの背中。木製デスクと簡素な椅子、世界の空気感が主役。
4. バーチャル小部屋での軽い仕事風景。木製デスクとミニマルチェア、背中のアニメキャラ。淡いUIパネルが漂い、時間感はぼかされている。
5. VRChatっぽいコワーキング空間、淡いネオンの光と浮くUI。木の小机と簡素な椅子、アニメキャラは自然体の作業姿勢。
6. 薄く光るUIのあるVR空間の小部屋。木製の小さな机とノートPC、背中を見せるアニメキャラが軽くタスクを進めている。

## Base 3: 生活感のあるミニマル
_必要最低限の家具、控えめな小物。木製デスクとミニマル椅子、背中のアニメキャラ。物語性を出さず、日常の空気感を重視。_
1. 必要最低限の家具だけがあるVR小部屋。木製デスクとミニマル椅子、背中を向けたアニメキャラがノートPCで作業。淡い室内光。
2. ミニマルなVRコワーキング空間。小さな木の机と簡素な椅子、アニメキャラの後ろ姿。落ち着いた日常の空気。
3. VRChatの小部屋での静かな仕事風景。木製デスクにノートPC、背中のアニメキャラ。時間帯のわからない柔らかな光。
4. ほとんど装飾のないVR空間の作業室。木の小机とミニマル椅子、アニメキャラの背中が控えめに作業中。
5. 生活感が控えめなVRコワーキング。木製の小さなデスクと簡素な椅子、アニメキャラの後ろ姿が穏やかに仕事。
6. ミニマルなVR小部屋での作業風景。木製デスク、ノートPC、背中を見せるアニメキャラ。派手さのない空気感。

4. 静かな反射のあるVR小部屋で仕事をするアニメキャラの背中。木製デスクとミニマル椅子、落ち着いた空気感。
5. 拡散光のやさしいVR作業室。木の小さな机と簡素な椅子、背中を向けるアニメキャラ。主役は空気感。
6. 柔らかな照明のVR空間で、木製デスクに向かうアニメキャラの後ろ姿。派手さのない、日常の仕事風景。
