# NimbleView（理想の“詰め込み”画像・動画ビューアー） for Windows

Windows エクスプローラーの「無駄スペースが多い」「サムネが絶妙に小さい」「画像と動画を行ったり来たりしたい」「1枚表示が虚無」「お気に入り欲しい」「動画はループして」「シークバーを映像に重ねるな」…を、**実利だけ**で殴り返すためのローカルビューアーです。

- 目的：**フォルダ内の画像/動画を、できるだけ大きいサムネで、できるだけ密に、できるだけストレスなく**眺める
- 方針：**UIの余白＝敵**。必要な機能は残し、邪魔なものは隠す

> 実装は Python + Qt（PySide6）。Windows向け。

---

## できること（MVPだけど「かゆい」を潰しに行ってる）

### グリッド（一覧）
- **サムネを大きく**（スライダーで即変更）
- **隙間を小さく**（密度優先）
- **拡張子バッジ**を右下に表示（JPG/PNG/MP4…で画像/動画が一瞬で区別できる）
- **ファイル名は非表示がデフォ**（ツールチップで全文 / 表示も切替可）
- **画像/動画を同じ一覧で混在表示**（切替はトグル）
- **検索フィルタ**（名前の部分一致）
- **お気に入り（★）**：Fキーでトグル、★だけ表示も可

### ビューアー（1枚/1本表示）
- 画像：**ズーム/パン**、ダブルクリックで Fit/100% 切替
- 左右に **前後プレビュー**（虚無の余白を“前後”に使う）
- 動画：**デフォルトでループON**、操作バーは **映像の下（非オーバーレイ）**
- 動画：**フレーム保存**（Cキー / ボタン）→ PNGで書き出し

---

## ショートカット（覚えると脳がラクになるやつ）
### 共通
- `F11`：フルスクリーン
- `Esc`：戻る / 閉じる

### グリッド
- `Enter` / ダブルクリック：ビューアーで開く
- `F`：お気に入り（★）トグル
- `Ctrl+O`：フォルダを開く
- `Alt+↑`：親フォルダへ
- `Ctrl+F`：検索へフォーカス
- `Esc`：検索クリア

### ビューアー
- `←` / `→`：前/次へ
- `F`：お気に入り（★）トグル
- `Space`：動画の再生/一時停止
- `L`：動画ループ切替
- `C`：動画フレーム保存（PNG）

## インストール（開発版）
1) Python 3.10+ を入れる  
2) 依存を入れる

```bash
pip install -r requirements.txt
```

3) 起動

```bash
python -m nimbleview
```

---

## （任意）動画サムネを“ちゃんと”出したい
デフォルトでも動画は「VIDバッジ + プレースホルダ」で区別できますが、  
**グリッドの動画サムネを先頭フレームにしたい**なら OpenCV を追加で入れてください。

```bash
pip install opencv-python
```

---

## exe化（Windows）
PyInstaller で固められます（環境で調整が必要なことがある）。

```bash
pip install pyinstaller
pyinstaller --noconsole --onefile -n NimbleView -m nimbleview
```

---

## 注意
- ファイルを削除/移動する機能は入れてません（“事故”を避けるため）。
- お気に入りはユーザーの AppData に JSON で保存します（元フォルダは汚しません）。
